## Minimum Qualifications

### All Interns
- **Python**: Functions, classes, decorators, list comprehensions
- **JavaScript/TypeScript**: ES6+, async/await, arrow functions  
- **Git**: Branch, commit, pull requests, merge conflicts
- **SQL**: SELECT, JOIN, INSERT, UPDATE, basic indexing
- **HTTP**: GET/POST/PUT/DELETE, status codes, JSON
- **Terminal**: Navigate, run scripts, basic commands

### Backend Interns (2-3)
- Built APIs with FastAPI, Flask, or Django
- Used SQLAlchemy or similar ORM
- Written SQL queries with JOINs
- Understand async/await in Python
- Used pytest or similar testing framework

### Frontend Interns (2)
- Built UIs with React (hooks, components, props)
- Used TypeScript (types, interfaces)
- Responsive CSS (Flexbox, Grid)
- Made API calls (fetch/axios)
- Used Git for team projects

### DevOps Intern (1)
- All backend + frontend requirements
- Docker (built/ran containers)
- CI/CD basics (GitHub Actions)
- Linux server basics (ssh, processes)
- Cloud basics (AWS EC2, S3)
